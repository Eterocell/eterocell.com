MIT 6.NULL: Lecture 6 - Version Control (Git)

发表于 2021-04-07 分类于 MIT 6.NULL

A Git repository’s data structure are basically objects and references.

A commit is immutable since its hash code will be altered if something is changed. Commit’s hash code is linked to its human readable name with basically a key-value map.

HEAD reference to your current status

master / main / trunk or something like that is the base branch where the developments happen in convention.

Fast-forward : when merging a branch that uses current HEAD as base and only contains new commits in itself, there is no need to create new commit of merging it.

MIT 6.NULL: Lecture 3 - Text Editor (Vim)

发表于 2021-04-07 分类于 MIT 6.NULL

Modes

  • normal mode: esc from other modes
  • insert mode: i from normal mode
  • replace mode: R from normal mode
  • selection mode: v from normal mode
    • visual-line mode: shift-v from normal mode
    • visual-block mode: ctrl-v from normal mode
  • command-line mode: : from normal mode

Commands

:q for quit

:qa for quit all

:w for write

:help for help

:help :w help for :w

:sp for separate windows for an identical file

Normal Mode

Navigation

j move down

k move up

h move right

l move left

w move forward by 1 word

b move backward by 1 word

e move towards the end of the word

0 move towards the beginning of the line

$ move towards the end of the line

^ move towards the first non-empty character on the line

ctrl+u scrolls up

ctrl+d scrolls down

G moves all the way down the buffer

gg moves all the way up

L moves to the bottom line showing on the screen

M moves to the middle line showing on the screen

H moves to the top line showing on the screen

fo moves to the first o after the cursor

Fo moves to the first o before the cursor

to moves to the character before the first o after the cursor

To moves to the character after the first o before the cursor

/ for searching

  • n goes to the next match

Editing

u for undo

ctrl+R for redo

i enters the insert mode

o opens a new line below and enters the insert mode

O opens a new line above and enters the insert mode

d deletion following a navigation command, for eg:

  • dw deletes a word

c similar to d but enters the insert mode

dd deletes the current line

cc deletes the current line and enters the insert mode

x deletes the current character

r following with a character to replace the original one

y copy

  • yy copies the current line
  • yw copies the current word

p paste

~ invert alphabets’ case

. repeats the last changes in the current cursor

Counting

for eg:

  • 4j does j command 4 times

Modifiers

i for inside

a for inside and the wrapper

for eg:

di( delete all contents in between ()

da( delete all contents between () including ()

v

Visual-line Mode

V

Select lines

Visual-block Mode

ctrl+v

Select rectangular blocks
