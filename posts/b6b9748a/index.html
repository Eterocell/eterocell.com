使用 VSCode 和 MinGW 实现 Windows C/C++ 开发

MinGW Distro

VSCode

官方文档

  • 下载并安装 VSCode

  • 在 VSCode 中安装 C/C++ 插件

  • 下载并安装 MinGW Distro，将其解压至 C:\Program Files\MinGW

  • 打开 cmd 输入以下命令添加环境变量

  • 重启 cmd 输入以下命令以检查环境变量是否正确添加
  • 正确添加环境变量后，打开 VSCode，选择存放 C/C++ 代码的文件夹，新建 HelloWorld 文件夹并在其中新建 HelloWorld.cpp 文件并保存
  • 在终端中输入以下命令创建工作区

  • 点击终端-运行生成任务，选择 C/C++:g++.exe build active file，你会发现 VSCode 在工作区目录中创建了 .vscode 文件夹以及其中的 tasks.json

  • 点击调试-启动调试，选择 C++ (GDB/LLDB)，选择 g++.exe build and debug active file，你会发现 VSCode 在 .vscode 文件夹中创建了 launch.json

  • 可以生成和调试之后，如果想更多修改 C/C++ 插件的设置，可以按 Ctrl+Shift+P 搜索 C/C++: 编辑配置，这将在 .vscode 文件夹中创建 c_cpp_properties.json 文件继而修改配置，修改 C/C++ 标准便是在这一步进行。

  • 安装 llvm 工具链，包括 clang, clangd, clang-tidy 等工具

  • 在 VSCode 中安装 clangd 插件

  • Enjoy
