[数学][水题] HDU 2674 N! Again

原题链接

HDU, Virtual Judge

题目

WhereIsHeroFrom: Zty, what are you doing ?
Zty: I want to calculate N!……
WhereIsHeroFrom: So easy! How big N is ?
Zty: 1 <=N <=1000000000000000000000000000000000000000000000…
WhereIsHeroFrom: Oh! You must be crazy! Are you Fa Shao?
Zty: No. I haven’s finished my saying. I just said I want to calculate N! mod 2009

Hint : 0! = 1, N! = N*(N-1)!

Input

Each line will contain one integer N(0 <= N<=10^9). Process to end of file.

Output

For each case, output N! mod 2009

Sample Input

4
5

Sample Output

24
120

原思路

已知 2009! 因为有因数 2009 => 2009! mod 2009 == 0。因而包括 2009! 及以后的阶乘模 2009 结果必为 0，则只需算 2009 之前的。

Code

#include <iostream>
int main() {
    long long num;
    while (std::cin >> num) {
        long long sum = 1;
        if (num < 2009) {
            for (long long i = 1; i <= num; i++) {
                sum *= i;
                sum %= 2009;
            }
            std::cout << sum << std::endl;
        }
        else {
            std::cout << 0 << std::endl;
        }
    }
    return 0;
}

改进思路

因为 2009 = 41 x 7 x 7，因而到 49 及以后的数的阶乘结果都将为 0.

改进后的 Code

#include <iostream>
int main() {
    long long num;
    while (std::cin >> num) {
        long long sum = 1;
        if (num < 49) {
            for (long long i = 1; i <= num; i++) {
                sum *= i;
                sum %= 2009;
            }
            std::cout << sum << std::endl;
        }
        else {
            std::cout << 0 << std::endl;
        }
    }
    return 0;
}
