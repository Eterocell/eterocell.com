Job Control

signal

SIGINT for signal interrupt, ctrl+c sends a SIGINT

SIGQUIT for signal quit, ctrl+\ sends a SIGQUIT

SIGKILL for signal kill, which kills the current process but leaves the possible orphaned processes.

SIGSTOP for signal suspend, which suspends the current action and leave it in background, ctrl+z sends a SIGSTOP

SIGHUP for signal hang up.

man signal for more signals manual (Available on Ubuntu WSL, and not on ArchWSL here)
